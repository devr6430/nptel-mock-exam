In digital image processing, quantization is a technique used to reduce the number of possible pixel intensity values. This is often done to compress an image or simplify its representation for analysis.

The following Python program demonstrates intensity quantization using the Python Imaging Library (PIL) and NumPy. The program loads a grayscale image, examines the intensity of each pixel, and replaces it with a new value based on which range it falls into. The intensity values are divided into 8 equal levels ranging from 0 to 255.



In the code above:
  • The image is loaded and converted to grayscale using .convert('L').
  • The image is processed pixel by pixel. The comparison is directly made using pixelMap[i, j].
  • The if-elif structure checks which of the eight 32-level bins the pixel belongs to.
  • A new pixel value from 0 to 7 is assigned depending on the range the original intensity falls into.
  • The quantized image is saved and loaded as a NumPy array for further processing or analysis.
This approach is a basic example of uniform quantization , where the intensity range (0–255) is equally divided and mapped to fewer levels.
